本ライブラリは河原康雄　著「関係の理論-Dedekind圏概説-」を主軸に、圏を拡張した関係代数の基本性質をまとめたものである。
・Dedekind圏
  L-関係の基本的な性質を持つ圏. Heything代数や写像合成などの構造を内包する.
・Schroder圏
  関係の二値性を持つDedekind圏. ブール代数の構造を内包する.
・Cantor圏
  圏の対象に冪や部分対象の概念を導入したもの. 集合論と非常に近い性質を持つ.


This library is based primarily on Yasuo Kawahara’s *Theory of Relations — An Introduction to Dedekind Categories*, and summarizes the fundamental properties of relation algebra as an extension of category theory.

* **Dedekind Category**
  A category equipped with the fundamental properties of L-relations. It incorporates structures such as Heyting algebras and relational composition.

* **Schröder Category**
  A Dedekind category with relational duality. It incorporates the structure of Boolean algebras.

* **Cantor Category**
  A category in which notions such as powers and subobjects are introduced on objects. It exhibits properties closely related to set theory.
